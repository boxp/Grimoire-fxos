# Grimoire-fxos

A Firefox twitter client written by clojure.

## Functions

- Post new tweet
- View UserStream

## Usage

- Write your consumer key & consumer secret to resources/consumerkey.clj

``` sh 
lein clean && lein cljsbuild once
```
